Listen 1.5

ul vs. ol

  • ul = unordered list
    • HTML tag ul
    • Listenelemente sind immer li
    • Beispiele
      1. To Do Listen
      2. Einkaufslisten
      3. Brainstorming Listen
      4. ...alle Listen bei denen die Reihenfolge keine Rolle spielt
  • ol = ordered list
    • HTML tag ol
    • vergibt den Listenpunten eine fortlaufende Nummerierung
    • Beispiele
      1. Inhaltsverzeichnis
      2. Ablaufpläne
      3. Schritt für Schritt Anleitungen
      4. Rankinglisten
